# Команда, таймлайн, правила

## Владельцы модулей

| Модуль | Владелец | Зависит от |
|---|---|---|
| `ingest/` + data-prep | Mid 2 | — |
| `extraction/` | Senior 2 | schemas, данные от ingest |
| `graph/` | Senior 1 | schemas, JSONL от extraction |
| `retrieval/` | Senior 1 | graph |
| `synthesis/` + `api/` | Strong | schemas (моки до готовности retrieval) |
| `frontend/` | Mid 1 | schemas/api.py (моки до готовности бэка) |
| Презентация, видео, сабмит | Mid 2 | демо от всех |

Правило: чужой модуль не трогаем без пинга владельца. Спорные вопросы
по контрактам решают Senior 1 + Senior 2 за 5 минут, без долгих обсуждений.

## Таймлайн 24 часа (Ч+0 = старт)

| Часы | Что происходит | Чекпоинт |
|---|---|---|
| 0–2 | Все: разбор корпуса, фиксация онтологии и контрактов, docker-compose up | Контракты заморожены |
| 2–8 | Параллельные ядра: extraction на 10–20 доках, схема Neo4j, скелет API, каркас UI на моках, прогон парсера по корпусу | Извлечение выдаёт валидный JSON |
| 8–14 | Retrieval + text2cypher + первый e2e-ответ | 1 эталонный запрос проходит e2e |
| 14–18 | Интеграция фронта с API, граф-визуализация, все 4 эталонных запроса | Демо кликается руками |
| 18–21 | «Вау»: пробелы знаний, противоречия, эксперты, экспорт. Mid 2 → презентация | Фичи в демо |
| 21–23 | CODE FREEZE. Деплой, видео ≤5 мин, презентация | Сабмит собран |
| 23–24 | Буфер. Сабмит загружен ЗА ЧАС до дедлайна | Сдано |

## Жёсткие правила
1. **Ч+2: контракты schemas заморожены.** Изменение после — только если
   без него демо не работает, и только через обоих сеньоров.
2. **Ч+21: code freeze.** После — только хотфиксы падений демо.
3. **Ч+14 — точка невозврата:** если e2e не работает, режем скоуп
   (выкидываем text2cypher, оставляем только векторный поиск), а не дебажим.
4. Всё, что не в demo-сценарии, — не приоритет. Спрашивай себя:
   «увидит ли это жюри за 5 минут питча?»
5. Сабмит грузим в Ч+23, не в 23:59 — платформы падают в последний час.

## Коммуникация
- Синки: 15 минут в Ч+2, Ч+8, Ч+14, Ч+18, Ч+21 — статус по чекпоинтам
- Блокер дольше 30 минут → эскалация сеньору, не сидим молча
