1. **Числовые диапазоны:** «Какие методы обессоливания воды применялись
   при содержании сульфатов 200–300 мг/л и сухом остатке ≤ 1000 мг/дм³?
   Приведи источники и условия.»
   Проверяет: NumericConstraint, text2cypher, фильтры, цитаты.
2. **Эксперименты по процессу:** «Найди эксперименты по кучному
   выщелачиванию никелевых руд в России после 2015 года.»
   Проверяет: фильтры гео+год, связи Experiment→Process→Material.
3. **Материал → свойства → режимы:** «Какие режимы обработки влияют
   на извлечение меди при флотации? Где источники противоречат друг другу?»
   Проверяет: operates_at_condition, contradicts, консенсус-секцию.
4. **Эксперты:** «Кто в компании / в литературе занимался очисткой
   сточных вод от сульфатов? Какие у них публикации?»
   Проверяет: authored_by, recommended_experts.

## Сценарий питча (5 июля, репетировать с таймером)
1. Боль (30 сек): литобзоры неделями, знания теряются с людьми
2. Живое демо (2.5 мин): запрос №1 с числами → ответ с цитатами
   и confidence → клик в граф → фильтр по географии → запрос №3
   с блоком противоречий
3. Инсайт (40 сек): карта пробелов знаний — «вот где нет ни одного
   эксперимента» + рекомендация экспертов
4. Архитектура (40 сек): 6 блоков, верифицируемость каждого факта,
   масштабирование до 1 млн сущностей и новых доменов
5. Roadmap (20 сек): роли/аудит, интеграция с внутренними хранилищами
Отвечает на вопрос тот, кто писал соответствующий модуль.
